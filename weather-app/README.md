# Weather App

Using the Weatherstack and Mapbox API's, pass a location to the app and recieve time/temp data about it.

```$ npm install```
```
$ node app.js Portland
//
// ->
// Currently in Portland Oregon:
//      Weather: Light Rain
//      Temp: 50℉ and feels like 50℉
//      Local time is: 11:50 PM
```
